title: Assigning Custom Attributes to Devices

createdAt: Wed Apr 22 2026 10:16:33 GMT+0000 (Coordinated Universal Time)

You can assign custom device attributes such as internal ID to one or more devices. Each attribute has a corresponding value that you can use for tasks like creating configurations and device groups. After you create custom attributes, you can assign them to devices. For more information about managing attributes, see [Attributes](Attributes.mdx).

**Procedure**

<WorkflowBlock>
  <WorkflowBlockItem>
    Log in to the Administrative portal.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Go to **Devices**.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Select one or more devices.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Click **Actions**.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Select **Assign Custom Attributes**.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Select *one* of the following options:
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    - Force assign (overwrite) all attributes even if any existing values are found.
    - Overwrite only if value is empty, and skip attributes with existing values.
      Select the attributes you want to assign and enter their values (empty values are not allowed).
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Click **Assign**.
  </WorkflowBlockItem>
</WorkflowBlock>

The **Custom Device Attributes** with their values can be exported to CSV format from the **Device Details** page.
