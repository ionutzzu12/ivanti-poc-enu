To setup Windows 10 Unified Endpoint Management (UEM):

1. Login at [https://portal.azure.com/](https://portal.azure.com/) as admin user, and select Entra ID.
2. Select "Mobility (MDM and MAM)" in the left panel and then click **+ Add Application**.

<Image src="Mobility.png" size="19" isUploading="false" initialPath="Mobility.png" githubPath="ENU/Mobility.png" position="flex-start" indent="2"></Image>

3. Select MobileIron\_UEM application from App gallery and click **Add**.

<Image src="MobileIron_EMM.png" size="16" isUploading="false" initialPath="MobileIron_EMM.png" githubPath="ENU/MobileIron_EMM.png" position="flex-start" indent="2"></Image>
