The Home screen layout configuration defines a layout of apps, folders, and web clips for the Home screen.

This section contains the following topics:

- [Defining a Home Screen Layout configuration](Home_Screen_Layout_Configuration.mdx)
- [Home Screen Layout Configuration settings](Home_Screen_Layout_Configuration.mdx)

**License**: Silver

**Eligible Devices**: iOS 9.3+ Supervised Only

## Defining a Home Screen Layout configuration

**Procedure**

1. Go to **Configurations** > click **+ Add**.
2. Type "home" in the search field, and then click the **Home Screen Layout** configuration. The Home Screen Layout Configuration details page opens.
3. :inlineImage[]{src="https://api.archbee.com/api/optimize/BbT7KQqqvz3LSYn2rwmao/zadJVm2L7WmGYx_tX0IeW_r36homescreenlayout.png" alt caption}
   Configure the settings on this page. Refer to the table in the section [Home\_Screen\_Layout\_Configuration\_Settings](Home_Screen_Layout_Configuration.mdx) for guidance on the values.
4. Click **Next** to configure the distribution settings. For shared iPad devices, select the **Device** channel or the **User** channel. For more information, see [Working with Configurations](Working_with_Configurations.mdx).
5. Click **Done**.
