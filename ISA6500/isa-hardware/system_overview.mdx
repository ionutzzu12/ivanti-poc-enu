## Introduction

The ISA6500 features a unique and highly optimized design for low wattage processor platforms. It is equipped with a high-efficiency low noise power supply. High-performance fans provide ample optimized cooling.

## LED Control Panel

The control panel located on the front of the ISA6500 chassis includes power control buttons and status LEDs. This section describes the LED indicators and the appropriate responses.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/WJ_1sqXGVJOuKYu_9rBf5_led-panel.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/led-panel.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/led-panel.png" showCaption="false"></Image>

### Control Panel Buttons

The control panel includes a power on/off button and a reset button.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/oT3K-NkWyOcyOQ6k7J7xD_isa6500hwpwr.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_pwr.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_pwr.png" showCaption="false"></Image>

**Power button**: The main power switch is used to apply or remove power from the power supply to the server system. To turn off the device, press the power button for four seconds. You must unplug the system before servicing.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/4k3i7b4MSVWCzbuAd15wY_isa6500hwreset.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_reset.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_reset.png" showCaption="false"></Image>

**Reset button**: The reset button is used to reboot the system.

### Control Panel LEDs

The control panel displays five LEDs to monitor the system.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/RcKS37kxPh5SEhsQybR57_isa6500hwcaution.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_caution.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_caution.png" showCaption="false"></Image>

**Information LED**: Alerts operator of several states, as noted in the table below:

| Status        | Description                                                                                    |
| ------------- | ---------------------------------------------------------------------------------------------- |
| Red OFF       | Indicates system is operating normally.                                                        |
| Red ON        | Indicates critical condition, may include:<br />- System fan failure<br />- System temperature |
| Red and Blink | Indicates non-critical condition, may include:<br />- Temperature and voltage issue            |

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/ELlEeqWVGeiXdXXprbMAV_isa6500hwnl1.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_nl1.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_nl1.png" showCaption="false"></Image>

**NIC1**: Associated with the Management port. It indicates network activity when flashing.

| LED   | Color | Description                                    |
| ----- | ----- | ---------------------------------------------- |
| On    | Blue  | Link between system and network or no access.  |
| Blink | Blue  | Data transmission or receiving is occurring    |
| Off   | N/A   | No data transmission or receiving is occurring |

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xoqoLyOYt-N6HM43f9jI9_isa6500hwnl2.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_nl2.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_nl2.png" showCaption="false"></Image>

**NIC2**: Unused. Will always be off.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/audNbJktVAfaPQUQGcfNu_isa6500hwhdd.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_hdd.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_hdd.png" showCaption="false"></Image>

**HDD**: Indicates SATA drive activity when flashing.

- Amber and ON: indicates HDD locate.
- Amber, ON and Blink: indicates HDD access.
- Off: No HDD access or no HDD fault.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/rAu1G6Z9Lucz0-6dg8046_isa6500hwpwrled.png" position="flex-start" size="40" initialPath="../resources/images/isa6500-hardware-guide-v1.0/isa6500hw_pwrled.png" githubPath="ISA6500/resources/images/isa6500-hardware-guide-v1.0/isa6500hw_pwrled.png" showCaption="false"></Image>

**Power**: Indicates power is being supplied to the system power supply. This LED should be illuminated when the system is operating.

- On and Blue: System is powered on.
- Blue and Blink: System is in sleep mode.
- Off: System is powered off or in hibernate mode.

The management port has two LEDs..

## INT and EXT Port LEDs

Two LAN ports are located on the NIC card. Each Ethernet port has two LEDs.

| LED           | Name                        | Color  | Status          | Description                             |
| ------------- | --------------------------- | ------ | --------------- | --------------------------------------- |
| 1             | 10 GbE Activity LED<br />   | N/A    | Off             | No link                                 |
|               |                             | Green  | Blinking On/Off | Actively transmitting or receiving data |
| 2<br /><br /> | <br />10 GbE Link LED<br /> | N/A    | Off             | No link                                 |
|               |                             | Yellow | On              | Linked at slower than 10 Gbps           |
|               |                             | Green  | On              | 10 Gbps link speed                      |

## Air Flow Direction

The following table gives the air flow direction of the power supply fan and internal CPU fan.

| Component        | Air Flow Direction      |
| ---------------- | ----------------------- |
| Power Supply Fan | To the rear of the unit |
| Internal CPU Fan | To the rear of the unit |
