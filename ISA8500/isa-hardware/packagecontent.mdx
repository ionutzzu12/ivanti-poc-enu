The ISA8500F/C package typically includes the following:

| Item                                     | Quantity |
| ---------------------------------------- | -------- |
| ISA8500C/F System                        | 1        |
| Cable, Console, Blue, RJ45 to DB9, 6 ft. | 1        |
| Cable Generic, C13, 1 Power Cord         | 2        |
| Cable, Power, C13/C14, 13A/125V, 1m      | 2        |
| Rackmount rail kit and brackets          | 1        |
| Packaging box, accessory kit             | 1        |
