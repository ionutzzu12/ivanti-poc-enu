## Introduction

The ISA8500 features a unique and highly optimized design for low wattage processor platforms. It is equipped with a high-efficiency low noise power supply. High-performance fans provide ample optimized cooling.

**SYS LED**: Alerts operator of several states, as noted in the table below:

| LED Color | Status | Description                                                                                                                            |
| --------- | ------ | -------------------------------------------------------------------------------------------------------------------------------------- |

| Amber     | Blink  | Non-critical condition, may indicate:&#xA;- Redundant power module failure&#xA;- Temperature and voltage issue&#xA;- Chassis intrusion |

## Management Port LEDs

The management port has two LEDs. The yellow LED indicates activity, while the link LED may be green, amber, or off to indicate the speed of the connection.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/_irI7QJJeNo-GUVpZMbMY_systemoverview18.png" position="flex-start" size="40" initialPath="../resources/images/isa8500-hardware-guide/system_overview_18.png" githubPath="ISA8500/resources/images/isa8500-hardware-guide/system_overview_18.png" showCaption="false"></Image>

| LED         | Name                                | Color      | Status | Description                                    |
| ----------- | ----------------------------------- | ---------- | ------ | ---------------------------------------------- |
| 1           | 1 GbE Speed LED&#xA;&#xA;           | Yellow     | On     | 1 Gbps data rate                               |
|             |                                     | Green      | On     | 100 Mbps data rate                             |
|             |                                     | N/A        | Off    | 10 Mbps data rate                              |
| 2&#xA;&#xA; | 1 GbE Link / Activity LED&#xA;&#xA; | Green&#xA; | On     | Link between system and network or no access   |
|             |                                     |            | Blink  | Data transmission or reception is occurring    |
|             |                                     | N/A        | Off    | No data transmission or reception is occurring |

## INT and EXT Port LEDs

Two LAN ports are located on the NIC card. Each Ethernet port has two LEDs.

| LED         | Name                      | Color  | Status          | Description                             |
| ----------- | ------------------------- | ------ | --------------- | --------------------------------------- |
| 1           | 10 GbE Activity LED&#xA;  | N/A    | Off             | No link                                 |
|             |                           | Green  | Blinking On/Off | Actively transmitting or receiving data |
| 2&#xA;&#xA; | &#xA;10 GbE Link LED&#xA; | N/A    | Off             | No link                                 |
|             |                           | Yellow | On              | Linked at slower than 10 Gbps           |
|             |                           | Green  | On              | 10 Gbps link speed                      |

The following table gives the air flow direction of the power supply fan and internal CPU fan.

| Component        | Air Flow Direction      |
| ---------------- | ----------------------- |
| Power Supply Fan | To the rear of the unit |
| Internal CPU Fan | To the rear of the unit |
